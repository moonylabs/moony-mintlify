## Get Plants

Retrieve a list of all plants.

<RequestExample>
```bash cURL
curl -X GET 'https://api.example.com/plants' \
  -H 'Authorization: Bearer YOUR_API_KEY'
```
</RequestExample>

<ResponseExample>
```json Success
{
  "plants": [
    {
      "id": 1,
      "name": "Rose",
      "color": "Red"
    }
  ]
}
```
</ResponseExample>
