## Delete Plant

Delete a plant from the system.

<RequestExample>
```bash cURL
curl -X DELETE 'https://api.example.com/plants/1' \
  -H 'Authorization: Bearer YOUR_API_KEY'
```
</RequestExample>

<ResponseExample>
```json Success
{
  "message": "Plant deleted successfully"
}
```
</ResponseExample>
