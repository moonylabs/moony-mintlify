## Navigation

Structure your documentation navigation in `docs.json`.

### Navigation Structure

```json
{
  "navigation": {
    "groups": [
      {
        "group": "Get Started",
        "pages": [
          "docs/welcome",
          "docs/quickstart"
        ]
      }
    ]
  }
}
```

<Info>
Navigation groups help organize related pages together.
</Info>
