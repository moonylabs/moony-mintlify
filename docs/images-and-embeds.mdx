## Images and Embeds

Learn how to add visual content to your documentation.

### Images

Add images using standard Markdown syntax:

```markdown
![Alt text](/path/to/image.png)
```

### Embeds

Embed external content using Mintlify components.

<Frame>
<iframe src="https://example.com" width="100%" height="400"></iframe>
</Frame>
