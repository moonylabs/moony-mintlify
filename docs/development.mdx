## Development Guide

Learn the best practices for developing and maintaining your documentation.

<Tip>
Use the Mintlify CLI for local development to preview changes instantly.
</Tip>

### Local Development

Run the development server to see your changes in real-time:

```bash
mintlify dev
```

<Info>
The dev server will automatically reload when you save changes to your files.
</Info>
