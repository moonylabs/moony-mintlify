## Create Plant

Create a new plant in the system.

<RequestExample>
```bash cURL
curl -X POST 'https://api.example.com/plants' \
  -H 'Authorization: Bearer YOUR_API_KEY' \
  -H 'Content-Type: application/json' \
  -d '{
    "name": "Tulip",
    "color": "Yellow"
  }'
```
</RequestExample>

<ResponseExample>
```json Success
{
  "id": 2,
  "name": "Tulip",
  "color": "Yellow",
  "created_at": "2024-01-15T10:30:00Z"
}
```
</ResponseExample>
