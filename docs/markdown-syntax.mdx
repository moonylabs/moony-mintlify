## Markdown Syntax

Mintlify supports standard Markdown syntax with additional components.

### Headings

Use headings to structure your content:

```markdown
## Heading 2
### Heading 3
#### Heading 4
```

### Lists

Create ordered and unordered lists:

```markdown
- Item 1
- Item 2
- Item 3

1. First item
2. Second item
3. Third item
```
