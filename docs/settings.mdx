## Settings

Configure your documentation through the `docs.json` file.

### Basic Configuration

```json
{
  "$schema": "https://mintlify.com/schema.json",
  "name": "Your Project",
  "theme": "maple"
}
```

<Tip>
Use the Mintlify schema to get autocomplete and validation in your editor.
</Tip>
