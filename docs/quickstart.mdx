## Quickstart Guide

Get started with Moony Labs in just a few steps.

<Steps>
  <Step title="Install dependencies">
    Install the required packages:

    ```bash
    npm install
    ```

    <Check>
    Your dependencies are installed successfully!
    </Check>
  </Step>

  <Step title="Configure your project">
    Set up your configuration file with your credentials.

    <Warning>
    Make sure to keep your credentials secure.
    </Warning>
  </Step>

  <Step title="Run your first command">
    Execute the following command:

    ```bash
    npm start
    ```

    <Check>
    If you see the welcome message, you're all set!
    </Check>
  </Step>
</Steps>
