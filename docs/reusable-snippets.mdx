## Reusable Snippets

Create reusable content that can be included across multiple pages.

<Tip>
Use snippets to maintain consistency across your documentation.
</Tip>

### Creating Snippets

Create a `.snippet` file in your docs directory:

```markdown
This is a reusable snippet that can be included in multiple pages.
```
