## Get Started

<Steps>
  <Step title="Download Flipcash">
    Download the Flipcash mobile app to access Moony. Flipcash provides the first public interface for interacting with the Moony Reserve and sending P2P payments.

    <Note>
    Available for iOS and Android. <a href="https://www.flipcash.com/download" target="_blank" rel="noopener noreferrer">Download Flipcash</a>
    </Note>
  </Step>

  <Step title="Access Moony">
    Once Flipcash is installed, use the link below to open the Moony profile directly in the app. From there you can view the current MNY price, acquire tokens, and send payments.

    <Note>
    Opening this link will launch Flipcash and navigate to the Moony profile. <a href="#">Access Moony in Flipcash</a>
    </Note>
  </Step>

  <Step title="Acquire MNY">
    Flipcash provides seamless onramp options directly in the app. Fund with a debit card or link an external wallet with USDC, then acquire MNY through the Moony Reserve at the current bonding curve price.

    <Info>
    Pricing is determined by a deterministic bonding curve. Each transaction updates pricing automatically according to the protocol's immutable rules.
    </Info>
  </Step>

  <Step title="Send P2P Payments">
    Send MNY directly to others or share payment links. Flipcash offers zero-fee P2P transfers that settle onchain with finality.

    <Check>
    You're ready to use Moony for P2P payments.
    </Check>
  </Step>
</Steps>
