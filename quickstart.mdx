## Quickstart Guide

Get started with your documentation in just a few steps.

<Steps>
  <Step title="Install dependencies">
    Install the Mintlify CLI:

    ```bash
    npm i -g mintlify
    ```

    <Check>
    CLI installed successfully!
    </Check>
  </Step>

  <Step title="Initialize your docs">
    Run the initialization command:

    ```bash
    mintlify init
    ```

    <Warning>
    Make sure you're in the directory where you want your docs.
    </Warning>
  </Step>

  <Step title="Start developing">
    Start the development server:

    ```bash
    mintlify dev
    ```

    <Check>
    Your docs are now running at localhost:3000!
    </Check>
  </Step>
</Steps>
